1.	Ensure that python and flask are installed in the computer. (pip install flask)
2.	After extract of zip folder, we need to CD to the folder that contains the file: “masapp.py”.
3.	In the command prompt in the directory containing “masapp.py”, type: SET FLASK_APP=masapp.py (enter)
4.	Then type: SET FLASK_DEBUG=1 (enter)
5.	Then type: python -m flask run (enter)
6.	Go to a CHROME browser and key in localhost:5000

7.	To exit the server, go to command prompt and enter Ctrl + C twice.
